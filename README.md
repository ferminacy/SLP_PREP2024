# SLPS' LEVEL OF PREPAREDNESS WHEN WORKING WITH INTERPRETERS
## ABOUT THIS PROJECT
Project evaluates the level of preparedness of SLPs when working with interpreters. Includes data files, analysis scripts, and documentation.
